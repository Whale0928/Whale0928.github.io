패션 ERP 솔루션
2022.08 ~ 재직

💡 네셔널, 네파, 야크, 스노우 피크, 바바 등의 아웃도어 브랜드들의
ERP 솔루션 구축 및 유지보수를 패션 ERP 솔루션 회사입니다.

부서 내 협업과 프로젝트 진행 상황 공유에는 Confluence를 활용했습니다.
이 도구를 통해 효율적인 일정 관리와 작업 공유를 실현하였고,

ERP 솔루션 구축과 유지보수를 통해 물류, 회계,사용자 비즈니스 프로세스를 이해하고,
프로젝트에서는 새로운 요구 사항 기능 구현, 외부 인터페이스 관련 작업, 시스템 개선 작업 등 다양한 역할을 수행했습니다.
회사내에서의 경험을 통해 ERP 솔루션의 전반적인 구조와 비즈니스 프로세스에 대한 이해를 하였고,
이를 통해 사용자의 요구사항을 이해하고, 요구사항에 맞는 기능을 구현하는 경험을 하였습니다.

📐기술 스택
Java Oracle JavaScript Confluence Git SVN Jenkins Maven Linux
💼 업무 경험

쿼리 최적화로 ERP 시스템의 응답 시간 개선
  • 단순 조회를 위해 불필요한 조회 중복 발생.
  • 500,000건 이상의 데이터 처리에 대한 쿼리 최적화.
  • 쿼리 한 번 실행 후 중복 조회 결과를 애플리케이션 서버 메모리에서 처리.
  • 사용자 경험 향상 및 DB 부하 감소로 서버 안정성 증가.

API 연동을 위한 공통 모듈 개발
  • ERP 솔루션과 Groupware 솔루션 간 API 전자결제 연동 기능 개발.
  • API 연동에 필요한 공통 모듈 개발.
  • 코드 중복 감소 및 유지보수 효율성 향상.
  • 타 파트에서도 공통된 모듈을 개발해 문제 대응에 효과적으로 대응.

사용자 중심의 기능 개선
  • 사용자와의 협의를 통한 개선사항 발견 및 적용.
  • DB 레이어에서의 강제 Round 처리 문제를 협의를 통해 사용자 입력 분배 비율로 해결.
  • 사용자와의 커뮤니케이션 강화 및 타협점 도출.

통계성 데이터 처리
  • 사업실적 부분 영역에서 통계성 데이터 처리 및 조회 경험.
  • 브랜드별 년매출, 품목별 년매출, 시즌별 목표 실적 등등 다양한 데이터 처리.
  • 사용자 요구사항 이해 및 요구사항에 맞는 기능 구현.
  • 과도한 범위의 조회를 위해 쿼리를 속도 향상을 위한 경험
