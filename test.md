    - 변동 사항이 많은 검색/ 정렬 조건을 처리하기 위해 도입 후 팀원에게 공유 전파

  - on-premise 환경에서의 prometheus , grafana를 활용한 모니터링 환경 구축

  - github action을 활용한 CI/CD 파이프라인 구축

  - 개인적으로 활용해본 github projects를 활용하여 <b>프로젝트 관리의 효율성 증가</b>
